Logic

Logic is the study of formal reasoning. The most basic element of logic is a statement. While statements in spoken language can be ambiguous in meaning, we will only work with statements that are strictly either true or false.

Statement A statement is a claim that is either true or false.

Truth Value A statement’s truth value indicates whether the statement is true or false.

An axiom is a statement that is simply assumed to be true, while a theorem is a statement that can be proven to be true.

Mathematics is based on the axiomatic method. We use defined terms alongside axioms assumed to be true in order to prove certain theorems. Different combinations of axioms can lead to different mathematical structures.

In English, we can combine statements into compound statements using conjunctions like “and” or “or”. The same idea can be used to combine logical statements.

Logical Connective Logical connectives combine statements to form compound statements.

Conjunction “P and Q” P ∧ Q
Disjunction “P or Q” P ∨ Q

Conditional Statement A conditional statement combines two statements using implication. P ⟹ Q

Some important things to note:

  • We can read this as “P implies Q” or “if P, then Q”.

  • P is called the hypothesis while Q is called the conclusion.

  • A false hypothesis can imply any conclusion, making the implication vacuously true.

  • The statement is false only when P is true and Q is false.

Let’s say we want to prove some conditional statement P ⟹ Q. We are only concerned with conditions that would make the statement false, namely if P is true and Q is false.

Proving a Conditional Statement There are three basic methods of proving a conditional statement P ⟹ Q.

1. Direct Proof Assume P is true. Show that Q would also be true.
2. Contrapositive Proof Assume Q is false. Show that P would also be false.
3. Proof by Contradiction Assume the negation of the conditional statement, P ⟹ ¬Q. Show that this assumption leads to an obvious contradiction.

Biconditional Statement A biconditional statement is a logical implication that goes both ways. P ⇔ Q

Functions

Consider the function f(x) = x2 for the natural numbers (1, 2, 3, …). f maps values as such: $$\begin{aligned} 1 &\mapsto 1 \\ 2 &\mapsto 4 \\ 3 &\mapsto 9 \\ \vdots \end{aligned}$$ The function f is actually a set of ordered pairs {(1,1), (2,4), (3,9), …} where we assign every natural number to its corresponding square.

Function Let X and Y be sets. A function from X to Y is a relation f ⊆ X × Y such that:

  1. for all x ∈ X, there exists some y ∈ Y such that (x,y) ∈ f, and

  2. for all x ∈ X and y1, y2 ∈ Y, if (x,y1) ∈ f and (x,y2) ∈ f, then y1 = y2.

The standard notation to define a function is f : X → Y where X is the domain and Y is the codomain of the function. We also write f(x) = y instead of (x,y) ∈ f.

Image Let f : X → Y be a function, and let A ⊆ X. The image of A is the set of all possible output values A can produce. f(A) ≔ {y∈Y:∃(x∈A)[y=f(x)]}

In a function f : X → Y, taking the image of X (i.e. f(X)) gives us the range of the function. It’s a subset of the codomain, which is not always equal to the codomain.

Inverse Image Let f : X → Y be a function, and let B ⊆ Y. The inverse image of B is the set of all values of X that map to something in B. f−1(B) ≔ {x∈X:f(x)∈B}

Notation of the image and preimage conflicts with our pre-existing idea of f(x) and f−1(x).

  • If we’re dealing with elements from the domain of f, we compute as we normally would.

  • If we’re dealing with subsets of the domain or codomain, we are working with images and preimages.

Open/Closed Interval An interval is a set that contains a range of elements.

  • Open Interval (a,b) = {n : a < n < b}

  • Closed Interval [a,b] = {n : a ≤ n ≤ b}

Image and Inverse Image Let f : ℝ → ℝ be defined by f(x) = x2. We have:

  • f({1,−1}) = {1}

  • f−1((0,1)) = (−1,0) ∪ (0,1)

  • f(f−1({−1})) = f(∅) = ∅

  • f−1(f({−1})) = f−1({1}) = { − 1, 1}

Field Axioms

Field A field is a set with two closed operations, addition and multiplication, that satisfy the following axioms for all a, b, c ∈ 𝔽:

Axiom Addition Multiplication
Associativity (a+b) + c = a + (b+c) (ab)c = a(bc)
Commutativity a + b = b + a ab = ba
Distributivity a(b+c) = ab + ac (a+b)c = ac + bc
Identities ∃(0∈𝔽)(a+0=a) ∃(1∈𝔽)(1≠0∧1a=a)
Inverses ∃(−a∈𝔽)(a+(−a)=0) (a≠0) ⇔ ∃(a−1∈𝔽)(aa−1=1)

“Closed operation” just means that we can add or multiply any two things in the field and still get a thing in the field. “Field” is a pretty broad term and encompasses a lot more than just the real numbers.

We may be tempted to start using familiar properties, like how multiplying by zero yields zero or how x−1−1 = x. However, at this stage, we only know of the five field axioms. We will have to prove most of the properties we’ve been familiar with, no matter how trivial they may seem.

Uniqueness of Addition in a Fieldadd-unique For any field 𝔽 and all a, b ∈ 𝔽, the equation a + x = b has a unique solution.

Proof. Let 𝔽 be a field, and let a, b ∈ 𝔽. We need to show both of the following:

  1. Existence: a + x = b has a solution x ∈ 𝔽.

  2. Uniqueness: if x1 and x2 are solutions to a + x = b, then x1 = x2.

Let x1 ≔ (−a) + b. Since  − a, b ∈ 𝔽 and 𝔽 is closed under addition, then x1 ∈ 𝔽. $$\begin{aligned} a + x_1 &= a + ((-a) + b) && x_1=(-a)+b \\ &= (a + (-a)) + b && \text{Additive Associativity} \\ &= 0 + b && \text{Additive Inverse} \\ &= b && \text{Additive Identity} \end{aligned}$$ Thus, x1 is a solution to a + x = b.

Suppose that there exists some x2 ∈ 𝔽 that is also a solution to a + x = b. Then a + x1 = b and a + x2 = b. $$\begin{aligned} a + x_1 &= a + x_2 \\ (-a) + (a + x_1) &= (-a) + (a + x_2) && \text{Add $-a$ to both sides} \\ ((-a) + a) + x_1 &= ((-a) + a) + x_2 && \text{Additive Associativity} \\ 0 + x_1 &= 0 + x_2 && \text{Additive Inverse} \\ x_1 &= x_2 && \text{Additive Identity} \end{aligned}$$ Therefore, a + x = b has only one solution. ◻

A similar proof can be done for multiplication where a ≠ 0.

Proving Equality in a Field Imagine we had elements a, b ∈ 𝔽 and had to prove they were equal. We can use the idea of uniqueness to prove that a = b.

  1. Create some equation in the form of x + y = z or x ⋅ y = z with y and z being fixed constants.

  2. Show that x = a and x = b are both valid solutions to the equation.

We can then use the fact that addition/multiplication is unique to conclude that a = b. Note: When using uniqueness of multiplication, we cannot let y = 0 since anything times zero is zero.

Multiplication by Zeromult-by-zero For any field 𝔽 and all a ∈ 𝔽, 0 ⋅ a = 0

Proof. Let 𝔽 be a field, and let a ∈ 𝔽. Consider the following equation: a + x = a

  1. Let x = 0. Then, by definition of the additive identity, a + 0 = a. Thus, x = 0 is a valid solution.

  2. Let x = a ⋅ 0. Then: $$\begin{aligned} a + (a \cdot 0) &= (a \cdot 1) + (a \cdot 0) \\ &= a (1 + 0) \\ &= a \cdot 1 \\ &= 1 \cdot a \end{aligned}$$ By definition of the multiplicative identity, x = a ⋅ 0 is also a valid solution.

Because uniqueness guarantees only one value for x, then it must follow that 0 = a ⋅ 0. ◻

Zero Product Property For any field 𝔽 and all a, b ∈ 𝔽, if ab = 0, then a = 0 or b = 0.

Proof. Let 𝔽 be a field, and let a, b ∈ 𝔽 where ab = 0.

  • If a = 0, we’re done.

  • If a ≠ 0, then a−1 ∈ 𝔽. Thus: $$\begin{aligned} ab = 0 &\implies aba^{-1} = 0a^{-1} && \text{Multiply both sides by}\ a^{-1} \\ &\implies aa^{-1}b = 0a^{-1} && \text{Multiplicative Commutativity} \\ &\implies 1b = 0a^{-1} && \text{Multiplicative Inverse} \\ &\implies b = 0a^{-1} && \text{Multiplicative Identity} \\ &\implies b = 0 && \text{\nameref{thm:mult-by-zero}} \end{aligned}$$

In either case, a = 0 or b = 0. ◻

Double Additive Inverse For any field 𝔽 and all a ∈ 𝔽, a =  − (−a).

Proof. Let 𝔽 be a field, and let a ∈ 𝔽. Consider the equation x + (−a) = 0.

  1. Let x = a. Then a + (−a) = 0 by definition of Additive Inverse.

  2. Let x =  − (−a). Then  − (−a) + (−a) = (−a) +  − (−a) = 0 by definition of Additive Inverse.

Because uniqueness of addition guarantees only one value for x, then it must follow that a =  − (−a). ◻

Double Multiplicative Inverse For any field 𝔽 and all a ∈ 𝔽 where a ≠ 0, a = (a−1)−1.

Proof. Let 𝔽 be a field, and let a ∈ 𝔽 where a ≠ 0. Consider the equation xa−1 = 1.

  1. Let x = a. Then aa−1 = 1 by definition of the Multiplicative Inverse.

  2. Let x = (a−1)−1. Then (a−1)−1a−1 = a−1(a−1)−1 = 1 by definition of the Multiplicative Inverse.

Because uniqueness of multiplication guarantees only one value for x, then it must follow that a = (a−1)−1. ◻

Order Axioms

In many fields, there is an ordering between the elements.

Ordered Field A field 𝔽 is ordered if and only if there is a relation < on 𝔽 such that all a, b, c ∈ 𝔽 satisfy the following axioms:

Axiom Description
Transitivity If a < b and b < c, then a < c
Trichotomy Exactly one is true: a < b or a = b or b < a
Additive Property If a < b, then a + c < b + c
Multiplicative Property If a < b and 0 < c, then ac < bc

This leaves us with the real numbers (ℝ) and rational numbers (ℚ). The complex numbers (ℂ) have no order between the elements. Is 1 + 2i < 2 + 1i, or 2 + 1i < 1 + 2i?

Order Notation and Terminology

  • a < b is equivalent to b > a

  • a ≤ b is equivalent to a < b or a = b

  • a ∈ 𝔽 is positive if a > 0

  • a ∈ 𝔽 is negative if a < 0

From now on, we will use basic algebra in proofs without explaining each individual step.

Be careful when negating inequalities!

  • The negation of a < b is a ≥ b, not a > b.

  • a < b < c means a < b and b < c, so its negation would be “a ≥ b or b ≥ c”

Negatives Flip Inequality Let 𝔽 be any ordered field, and let a ∈ 𝔽. If a < 0, then  − a > 0.

Proof. Let 𝔽 be an ordered field, and let a ∈ 𝔽 where a < 0. By the additive property, we can add  − a to both sides of the inequality. Then (−a) + a < (−a) + 0 which simplifies to 0 <  − a. Therefore,  − a > 0. ◻

0 < 1 0 < 1 in any ordered field.

Proof. Suppose for contradiction that 1 < 0. Then  − 1 > 0, so we can apply the multiplicative property to 0 <  − 1 as such: $$\begin{aligned} 0 (-1) &< (-1)(-1) \\ &= -(-1) \\ &= 1 \end{aligned}$$ That is, 0 < 1, which contradicts our supposition that 1 < 0. Thus, it must be true that 0 ≤ 1 (the negation of 1 < 0). Note that from the field axioms, we know 0 ≠ 1. Therefore, 0 < 1. ◻

Completeness

Completeness deals with the idea of existence of bounds.

Upper Bound M ∈ 𝔽 is an upper bound for A if M ≥ x for all x ∈ A.

Lower Bound M ∈ 𝔽 is a lower bound for A if M ≤ x for all x ∈ A.

We say a set is bounded above if there exists an upper bound for that set. Similarly, we say a set is bounded below if there exists a lower bound for that set.

Supremum s ∈ 𝔽 is a supremum of A (sup A) if:

  1. s is an upper bound for A, and

  2. s ≤ M for all upper bounds M of A.

Infimum s ∈ 𝔽 is an infimum of A (inf A) if:

  1. s is a lower bound for A, and

  2. s ≥ M for all lower bounds M of A.

Maximum s ∈ 𝔽 is a maximum of A (max A) if s is a supremum of A and s ∈ A.

Minimum i ∈ 𝔽 is a minimum of A (min A) if i is an infimum of A and i ∈ A.

Now we can finally tackle the notion of “completeness”.

Complete Fieldcomplete-field An ordered field 𝔽 is complete if every A ⊆ 𝔽 that is bounded above has a supremum.

Real Numbers (ℝ) The set of real numbers is the unique complete ordered field.

The rational numbers are not complete. Consider the following set: A ≔ {r ∈ ℚ : r2 < 2} ⊂ ℚ A natural choice for a supremum might be $\sqrt{2}$, but note that $\sqrt{2}$ is not a rational number (proven in Example [ex:sqrt-2-irrational]).

Approximation Property for the Supremum Let A ⊆ ℝ be nonempty and bounded above. Then s = sup A if and only if:

  1. ∀(x∈A)(s≥x)

Parity Parity describes whether an integer is even or odd. For any integer n ∈ ℤ:

  • n is even if n = 2k for some k ∈ ℤ

  • n is odd if n = 2k + 1 for some k ∈ ℤ

Every integer is either even or odd Every integer must be either even or odd, never both.

Proof. First, consider the parity of 0 and 1:

  1. 0 is even since 0 = 2 ⋅ 0

  2. 1 is odd since 1 = 2 ⋅ 0 + 1

Next, let’s use induction for integers greater than 0. Assume that n ∈ ℕ is either even or odd.

  1. If n is even, then n = 2k for some k ∈ ℤ. Then, n + 1 = 2k + 1, so n + 1 is odd.

  2. If n is odd, then n = 2k + 1 for some k ∈ ℤ. Then, n + 1 = (2k+1) + 1 = 2k + 2 = 2(k+1), so n + 1 is odd.

Now, let’s prove the same for negative numbers. If n ∈  − ℕ, then  − n ∈ ℕ, so  − n is either even or odd.

  1. If  − n is even, then  − n = 2k for some k ∈ ℤ. Then n = 2(−k) where (−k) ∈ ℤ, so n is even.

  2. If  − n is odd, then  − n = 2k + 1 for some k ∈ ℤ. Then n =  − (2k+1) = 2(−k) − 1 = 2(−k−1) + 1. Since (−k−1) ∈ ℤ, then n is odd.

This completes the proof. ◻

Addition/Multiplication of Odd Numbers Prove that the sum of two odd integers is even, and the product of two odd integers is odd.

Proof. Let m, n ∈ ℤ be odd integers. Then m = 2k + 1 and n = 2l + 1 for some m, n ∈ ℤ. Then: $$\begin{aligned} m+n &= (2k+1) + (2l+1) \\ &= 2k+2l+2 \\ &= 2(k+l+1) \end{aligned}$$ Since k + l + 1 ∈ ℤ, then m + n is even. Similarly: $$\begin{aligned} mn &= (2k+1)(2l+1) \\ &= 4kl + 2k + 2l + 1 \\ &= 2(kl + k + l) + 1 \end{aligned}$$ Since kl + k + l ∈ ℤ, then mn is odd. ◻

Induction on Odd Integers Prove that 7n + 13n is divisible by 5 for all odd n ∈ ℕ.

Proof. We will use induction on k ∈ ℤ were n = 2k + 1. First, if k = 0: 7n + 13n = 72 ⋅ 0 + 1 + 132 ⋅ 0 + 1 = 71 + 131 = 20 = 5(4) That is, 5 ∣ (71+131).

Next, Suppose that 5 ∣ (72k + 1+132k + 1) for some k ∈ ℤ where k ≥ 0. Then: $$\begin{aligned} 7^{2(k+1)+1} + 13^{2(k+1)+1} &= 7^{2k+3} + 13^{2k+3} \\ &= 7^2 + 7^{2k+1} \\ &= 7^2 \cdot 7^{2k+1} + 13^2 \cdot 13^{2k+1} \\ &= 49 \cdot 7^{2k+1} + 169 \cdot 13^{2k+1} \\ &= 49 (7^{2k+1} + 13^{2k+1}) + 120 \cdot 13^{2k+1} \end{aligned}$$ By the induction assumption, we know that 72k + 1 + 132k + 1 = 5c for some c ∈ ℤ. Thus: $$\begin{aligned} 7^{2k+3} + 13^{2k+3} &= 49(7^{2k+1} + 13^{2k+1}) + 120 \cdot 13^{2k+1} \\ &= 49(5c) + 5(24 \cdot 13^{2k+1}) \\ &= 5(49c + 24 \cdot 13^{2k+1}) \end{aligned}$$ Because (49c+24⋅132k + 1) ∈ ℤ, then 5 ∣ (72k + 3+132k + 3). Greatest Common Divisor Given a, b ∈ ℤ, the greatest common divisor of a and b is (informally) the largest integer d that divides both a and b. d = gcd (a,b)

Greatest Common Divisors If a, b ∈ ℤ, then there is a common divisor d of a and b such that:

  1. d = am + bn for some mn ∈ ℤ

  2. d is non-negative

  3. every common divisor of a and b divides d

  4. if a and b are not both zero, and c is a common divisor of a and b, then c ≤ d

If a and b are not both zero, then the divisor d is called the greatest common divisor of a and b, denoted d = gcd (a,b).

Proof. We will first prove that d exists when both a, b ≥ 0. Let’s assume without loss of generality that a ≤ b (i.e. if a ≰ b, swap the two and continue the proof).

  1. If a = 0, then b ∣ a and b ∣ b, so b is a common divisor of a and b. Then, we can choose d as a linear combination of a and b: d = 0 ⋅ a + 1 ⋅ b = b Let a + b = k, and let P(k) be the statement: “If a + b = k, then there is a common divisor d of a and b such that d = ma + nb for mn ∈ ℤ”

    Base Case: If k = 0, then a = b = 0, so P(0) is true from above.

    S.I. Hypothesis: Suppose that P(k) is true for 0 ≤ k ≤ n where n ≥ 0 ∈ ℤ.

    Let a, b ≥ 0 ∈ ℤ such that a + b = n + 1. If a = 0, then we are done. Otherwise, 1 ≤ a ≤ b, so 0 ≤ b − a < b. Let c = b − a. Then: a + c = a + b − a = n + 1 − a ≤ n By our induction hypothesis, there is a common divisor d of a and c, and there exist m, n ∈ ℤ such that: $$\begin{aligned} d &= ma+nc \\ &= ma + n(b-a) \\ &= (m-n)a + nb \end{aligned}$$ Because d is the common divisor of a and b − a, then d also divides any linear combination of a and b − a, primarily a + (b−a) = b. Thus d is a common divisor of a and b, and d = am + bn for some m, n ∈ ℤ. That is, P(n+1) is true by strong induction. In particular, property 1 holds.

  2. If the common divisor d from part 1 is negative, then  − d > 0 is also a common divisor of a and b. Also,  − d = a(−m) + b(−n) where  − m,  − n ∈ ℤ. Thus, we can assume that d ≥ 0.

  3. Let c be a common divisor of a and b. Then c ∣ (am+bn) for all m, n ∈ ℤ, so in particular, c ∣ d.

  4. Suppose that at least one of a and b is not zero. Let c be a common divisor of a and b. Then c ≠ 0 (otherwise, a = b = 0).

    • If c < 0, then c < d because d ≥ 0.

    • If c > 0, then a and b are both non-negative, and d > 0. Consequently, c, d ∈ ℕ. We know c ∣ d from part 3, so c ≤ d.

 ◻

Division Algorithm For all real numbers 0 < d < a, there exists a unique integer q (the quotient) and a unique real number r (the remainder) such that 0 ≤ r < d and a = qd + r

Proof. Let E = {m∈ℤ:md≤a}. We know E is nonempty since both 0 and 1 must be in this set. Also, E bounded above by $\nicefrac{a}{d}$. Therefore, by the for ℤ, E has a maximum. Let q ≔ max E, so qd ≤ a. Let r = a − qd ≥ 0. Because q + 1 ∉ E, we know that (q+1)d > a, so qd + d > a, so qd + d > a, and thus r = a − qd < d. That is, 0 ≤ r < d and a = qd + r. This proves the existence of q and r.

To prove uniqueness, assume that q′ and r′ satisfy the Division Algorithm. That is: 0 ≤ r′ < d  and  a = q′d + r′ We need to show q = q′ and r = r′. Assume without loss of generality that q′ ≤ q. Thus: (q−q′)d = qd − q′d = (a−r) − (a−r′) = r′ − r (The first a = qd + r and the second a = q′d + r′). We know r ≥ 0 and r′ < d, so r′ − r < d. Also, d ∣ (r′−r), so either d ≤ r′ − r which is a contradiction. Thus, r′ − r = 0, so r′ = r. Thus (q−q′)d = 0 and d ≠ 0, so q − q′ = 0. That is, q = q′. Therefore, the quotient ad the remainder in the division algorithm are unique. ◻

A more general version where we allow a ≤ 0 is the following:

“If a, d ∈ R where d ≠ 0, then there exists a unique q ∈ ℤ and a unique r ∈ R such that $0 \leq r < \abs{d}$ and a = qd + r”.

Using Uniqueness of the Division Algorithm Prove that if a, b ∈ ℕ where a, b > 1, then ab + 1 is not divisible by a or b.

Proof. Suppose for contradiction that a ∣ (ab+1). Then there exists c ∈ ℤ where ab + 1 = ac. This violates the uniqueness of the division algorithm. Let m ≔ ab + 1. Then: $$m = \underbracket{b}_{\text{quotient}} \cdot a + \underbracket{1}_{\text{remainder}}$$ $$m=\underbracket{c}_{\text{quotient}} \cdot a + \underbracket{0}_{\text{remainder}}$$ Because we have two different remainders when dividing m by a, this violates the uniqueness of the division algorithm. Therefore, ab + 1 is not divisible by a.

A similar proof shows ab + 1 is not divisible by b. ◻

Prime Factorization For a natural number a ≥ 2, a prime factorization of a is a product a = p1p2⋯pk where each pi is prime and p1 ≤ p2 ≤ ⋯ ≤ pk.

Ordering on the primes guarantees that each natural number n ≥ 2 has a unique prime factorization. The prime numbers themselves are not necessarily distinct.

Existence of Prime Factorization Every natural number can be written as a product of primes.

Proof. Note that n = 1 is an empty product. Also, n = 2 is prime, so 2 can be written as a product of primes.

Assume that for some n ≥ 2, every natural number 2 ≤ k ≤ n can be written as a product of primes.

  • If n + 1 is prime, then n + 1 is automatically a product of primes.

  • If n + 1 is not prime, then n + 1 = ab where a and b are proper divisors of n + 1 (i.e. 2 ≤ a ≤ n and 2 ≤ b ≤ n). By the induction hypothesis, each of a and b can be written as a product of primes. a = p1p2⋯pk b = q1q2⋯ql Hence, n + 1 = ab = (p1p2⋯pk)(q1q2⋯ql), so n + 1 is a product of primes.

By principle of strong induction, our theorem holds. ◻

Fundamental Theorem of Arithmetic Every natural number a ≥ 2 has a unique prime factorization.

Proof. Let a ∈ ℕ where a ≥ 2. We know a prime factorization for a exists from our previous proof. If some natural number does not have a unique prime factorization, then by the there exists a smallest such number. Assume a ≥ 2 is the smallest natural number with more than 1 prime factorization. That is, we can write a as: $$\begin{aligned} a = p_1 p_2 \cdots p_k \quad &\text{where} \quad p_1 \leq p_2 \leq \cdots \leq p_k \quad \text{and} \\ a = q_1 q_2 \cdots q_l \quad &\text{where} \quad q_1 \leq q_2 \leq \cdots \leq q_l \end{aligned}$$ where pi ≠ qi for some i ∈ ℤ. Now we need to show that these two prime factorizations are different. Assume for contradiction that p1 = q1. Then, p2p3⋯pk = q2q3⋯ql < a. We then have a natural number smaller than a with two distinct prime factorizations. Hence, this contradicts our choice of a as the smallest such number. Therefore, we know p1 ≠ q1.

Next, without loss of generality, we can assume that p1 < q1 (i.e. if p1 > q1, swap all pi with qi and continue). By the Division Algorithm, we can write q1 = qp1 + r where q1r ∈ ℕ and 0 < r < p1. (If r = 0, then q1 = qp1, giving q1 a proper divisor, so r ≠ 0.)

Let w = rq2q3⋯ql < a (because r < p1 < q1). Because r < p1 and p1 < qi for all i ∈ ℤ such that 2 ≤ i ≤ l, then the prime p1 does not appear in this product. But: $$\begin{aligned} w &= r q_2 q_3 \cdots q_l \\ &= (q_1 - qp_1) q_2 q_3 \cdots q_l \\ &= q_1 q_2 \cdots q_l - q p_1 q_2 q_3 \cdots q_l \\ &= p_1 p_2 \cdots p_k - q p_1 q_2 q_3 \cdots q_l \\ &= p_1 (p_2 p_3 \cdots p_k - q\ q_2 q_3 \cdots q_l) \end{aligned}$$ Thus, by writing p2p3⋯pk − qq2q3⋯ql as a product of primes, we will have a prime factorization of w that contains p1. That is, w < a has two distinct prime factorizations. This contradicts a being the smallest number with two distinct prime factorizations. Therefore, every n ∈ ℕ where n ≥ 2 has a unique prime factorization. ◻

Consequences of Unique Prime Factorizations

  • If a ∈ ℕ and p is a prime such that p ∣ a, then p is in the prime factorization of a.

  • If a, b ∈ ℕ and a, b ≥ 2, then the factors in the prime factorization of ab consist of precisely one prime for each instance of a prime in the prime factorizations of a and b.

  • Suppose that p ∈ ℕ where p ≥ 2. Then p is prime if and only if the following property holds: “If a, b ∈ ℕ and p ∣ ab, then p ∣ a or p ∣ b. In particular, if p ∣ a2 then p ∣ a”.

One-to-one and Onto Functions

One-to-one Function A function f : X → Y is one-to-one (injective) if: ∀(x1,x2∈X)[f(x1)=f(x2)⟹x1=x2]

Note that the converse statement x1 = x2 ⟹ f(x1) = f(x2) is automatically true by definition of a function.

Onto Function A function f : X → Y is onto (surjective) if: ∀(y∈Y)∃(x∈X)(f(x)=y)

Bijective Function A function f : X → Y is bijective if f is both one-to-one and onto.

f : X → Y is an onto function if and only if f(X) = Y

Proof. First, suppose f : X → Y is onto. By definition of a function, f(X) ⊆ Y, so we need to show Y ⊆ X. Let y ∈ Y be arbitrary. Because f is onto, then there exists x ∈ X such that f(x) = y. That is, y ∈ f(X), so Y ⊆ f(X). Therefore, f(X) = Y.

Next, suppose f(X) = Y, and let y ∈ Y be arbitrary. Then y ∈ f(X), so y = f(x) for some x ∈ X. Because this is true for all y ∈ Y, then f is an onto function. ◻

If f : X → Y is a one-to-one function, then f(A∩B) = f(A) ∩ f(B) for all A, B ⊆ X.

Proof. From section 1.3, we know that f(A∩B) ⊆ f(A) ∩ f(B). Suppose f is one-to-one. Let y ∈ f(A) ∩ f(B). That is, y ∈ f(A) and y ∈ f(B). Thus, there exists some x1 ∈ A such that f(x1) = y, and there exists some x2 ∈ B such that f(x2) = y. That is, f(x1) = f(x2). Because f is one-to-one, that means x1 = x2. Hence, x1 ∈ A ∩ B, so y = f(x1) ∈ f(A∩B). Therefore, f(A) ∩ f(B) ⊆ f(A∩B). ◻

f : X → Y is an onto function if and only if f(f−1(B)) = B for all B ⊆ Y.

Proof. First, assume f is an onto function. From section 1.3, we know that f(f−1(B)) ⊆ B for all B ⊆ Y. Let y ∈ B. Because f is onto, there exists x ∈ X such that f(x) = y. Because y ∈ B, then x ∈ f−1(B). Thus, we can apply the image to both sides to attain y = f(x) ∈ f(f−1(B)). Therefore, B ⊆ f(f−1(B)), so B = f(f−1(B)).

Next, assume that f(f−1(B)) = B for all B ⊆ Y. Let y ∈ Y be arbitrary. If y ∈ B, then by our initial assumption, y ∈ f(f−1(B)). Thus, there exists x ∈ f−1(B) such that f(x) = y. But then x ∈ X and f(x) = y, so f is onto. ◻

f : X → Y is one-to-one if and only if f−1(f(A)) = A for all A ⊆ X.

Proof. Assume f is one-to-one. From section 1.3, we know that A ⊆ f−1(f(A)) for all A ⊆ X. Let x1 ∈ f−1(f(A)). Then f(x1) ∈ f(A), so there exists x2 ∈ A such that f(x1) = f(x2). Because f is one-to-one, then x1 = x2. Consequently, x1 ∈ A. Therefore, f−1(f(A)) ⊆ A, so f−1(f(A)) = A.

Suppose that f−1(f(A)) = A for all A ⊆ X. We need to show that as a consequence, f is a one-to-one function. Let x1, x2 ∈ X such that f(x1) = f(x2). Let A = {x1}. By assumption: $$\begin{aligned} f^{-1}(f(A)) &= \{ x \in X : f(x) \in f(A) \} \\ &= \{ x \in X : f(x) = f(x_1) \} \end{aligned}$$ Because x2 ∈ X and f(x2) = f(x1), then x2 ∈ f−1(f(A)). By our initial assumption, we also have x2 ∈ A. However, we defined A as having only one distinct element. As such, x1 = x2 and f(x1) = f(x2), satisfying the definition of a one-to-one function. ◻

Inverse Function Let f : X → Y be a function. A function g : Y → X is an inverse of f if:

  1. ∀(y∈Y)(f(g(y))=y)

  2. ∀(x∈X)(g(f(x))=x)

The standard notation for the inverse of f : X → Y is f−1. This is consistent with our earlier definition of f−1(B) as the inverse image of B ⊆ Y. It is possible that the inverse of a function does not exist, but we can still apply the inverse image to subsets of the co-domain.

Show that if g is an inverse of f, then f is an inverse of g

Proof. Because g is an inverse function of f, we know that g : Y → X satisfies f(g(y)) = y for all y ∈ Y, and g(f(x)) = x for every x ∈ X. But then f : X → Y satisfies g(f(x)) = x for x ∈ X, and f(g(y)) = y for all y ∈ Y. That is, f is an inverse of g. ◻

Existence of Inverse Functions Let f : X → Y be a function. Then f has an inverse if and only if f is bijective.

Proof. We will need to prove both directions of the logical equivalence. First, let’s assume that f : X → Y has an inverse function called g : Y → X. Then:

  • ∀(y∈Y)[f(g(y))=y], and

  • ∀(x∈X)[g(f(x))=x]

We need to show f is both one-to-one and onto. Let x1, x2 ∈ X such that f(x1) = f(x2). Then: $$\begin{aligned} x_1 &= g(f(x_1)) \\ &= g(f(x_2)) \\ &= x_2 \end{aligned}$$ Thus, f is a one-to-one function.

Let y ∈ Y. Then y = f(g(y)) and g(y) ∈ X. Thus, f is an onto function. Since f is both one-to-one and onto, then f is a bijective function.

Next, we need to prove the converse statement. Suppose that f is a bijective function. Then f is onto, so for any y ∈ Y there exists x ∈ X such that y = f(x). Define g : Y → X by $\underbrace{g(y)=x\ \text{if and only if}\ y=f(x)}_{\text{only works for one-to-one}}$. That is:

  1. If f(x) = y, then g(f(x)) = g(y) = x for all x ∈ X.

  2. If g(y) = x, then f(g(y)) = f(x) = y for all y ∈ Y.

Therefore, g is an inverse function of f. ◻

Uniqueness of Inverse Functions If f : X → Y has an inverse, then the inverse is unique. Suppose that g1 : Y → X and g2 : Y → X are both inverse functions of f : X → Y. We need to prove g1 = g2 by showing that for every y ∈ Y g1(y) = g2(y). Let y ∈ Y. Because f is onto, there is some x ∈ X such that y = f(x). Then: $$\begin{aligned} g_1(y) &= g_1(f(x)) \\ &= x \\ &= g_2(f(x)) \\ &= g_2(y) \end{aligned}$$ Therefore, g1 = g2, so f has a unique inverse.

Let f : X → Y be surjective (onto), and let g : Y → X satisfy g(f(x)) = x for all x ∈ X. Show that f is bijective and that g = f−1.

Proof. We already know that f is onto, so we only need to show that f is one-to-one. Let x1, x2 ∈ X such that f(x1) = f(x2). Then: $$\begin{aligned} x_1 &= g(f(x_1)) \\ &= g(f(x_2)) \\ &= x_2 \end{aligned}$$ Thus, f is one-to-one and is therefore a bijective function.

Next, we need to show g = f−1 (i.e. f(g(y)) = y for all y ∈ Y). Note that f is onto, so for any y ∈ Y there exists x ∈ X such that f(x) = y. Therefore: $$\begin{aligned} f(g(y)) &= f(g(f(x))) \\ &= f(x) \\ &= y \end{aligned}$$ Thus, g is the inverse function of f. ◻

Composition If f : X → Y and g : Y → Z are functions, then the composition of f and g is the function g ∘ f : X → Z where (g∘f)(x) = g(f(x)).

Suppose that f : X → Y and g : Y → Z are functions such that g is one-to-one and g ∘ f is onto. Show that f is also onto.

Proof. Let y ∈ Y. Then g(y) = z for some z ∈ Z. Because g ∘ f is onto, there exists x ∈ X such that (g∘f)(x) = g(f(x)) = z. That is, g(y) = g(f(x)). Because g is one-to-one, we have y = f(x) where x ∈ X. Therefore, f is an onto function. ◻

Composition Preserves Injectivity Show that if f : X → Y and g : Y → Z are both one-to-one, then g ∘ f is one-to-one.

Proof. Let x1, x2 ∈ X such that (g∘f)(x1) = (g∘f)(x2). That is, g(f(x1)) = g(f(x2)). Because g is one-to-one, we have f(x1) = f(x2). Because f is one-to-one, we have x1 = x2. Therefore, g ∘ f is one-to-one. ◻

Composition Preserves Surjectivity Suppose that f : X → Y and g : Y → Z are functions such that g ∘ f is one-to-one. We can prove f is one-to-one.

Proof. Let x1, x2 ∈ X such that f(x1) = f(x2). Then g(f(x1)) = g(f(x2)), so (g∘f)(x1) = (g∘f)(x2). Because g ∘ f is one-to-one, then x1 = x2. Therefore, f is one-to-one. ◻

However, g is not necessarily one-to-one.

Identity Function For a non-empty set A, the identity function on A is the function idA : A → A defined by: ∀(a∈A)(idA(a)=a)

This allows us to rephrase the definition of an inverse function as follows:

Let f : X → Y be a function. A function g : Y → X is an inverse of f if: g ∘ f = idX  and  f ∘ g = idY

Equivalence Relations

Recall that for two sets X and Y the Cartesian product X × Y is the set of ordered pairs where X × Y = {(x,y):x∈X and y∈Y}

Relation A relation on X × Y is any subset of X × Y. A relation on X is any subset of X × X.

Note that any function is also a relation. Recall the formal definition of a function f: f = {(x,y) : y = f(x)} ⊆ X × Y We use a special notation for functions where (x,y) ∈ f can be written as y = f(x). Non-function relations also have a special notation: $$(x,y) \in R \iff \underbrace{x \mathrel{R} y}_{\mathclap{\text{``$x$ is related to $y$''}}}$$ That is, R = {(x,y) : x R y}.

Let L ≔ {(x,y) : x, y ∈ ℝ, x < y} ⊆ ℝ2. Then: (x,y) ∈ L ⇔ x L y ⇔ x < y

Equivalence Relation Let X be a non-empty set, and let R be a relation on X. R is an equivalence relation if:

1. R is reflexive ∀(x∈X)(xRx)
2. R is symmetric x R y ⟹ y R x
3. R is transitive x R y ∧ y Z z ⟹ x R z

An equivalence relation is often denoted using the symbol ∼.

Equivalence Relations

  1. R = {(a,b) ∈ ℕ2 : a ∣ b}

    a|a for all a ∈ ℕ, so R is reflexive. Also, if a ∣ b and b ∣ c, then a ∣ c, so R is transitive. However, 2 ∣ 4 but 4 ∤ 2, so R is not symmetric. Therefore, R is not an equivalence relation.

  2. For x, y ∈ ℝ, x R y if x2 = y2

    x2 = x2 so x R x and thus R is reflexive. Also, if x2 = y2 then y2 = x2, so R is symmetric. And, if x2 = y2 and y2 = z2 then x2 = z2, so R is also transitive. Therefore, R is an equivalence relation.

Reflexivity, Symmetry, and Transitivity For x, y ∈ ℝ, define x R y to mean that $\abs{x-y} < 1$

  1. $\abs{x - x} = 0 < 1$, so R is reflexive.

  2. $\abs{x-y} = \abs{y-x}$, so if $\abs{x-y} < 1$ then $\abs{y - x} < 1$. That is, R is symmetric.

  3. $\abs{0 - 0.5} = 0.5$ so 0 R 0.5, and $\abs{0.5 - 1} = 0.5 < 1$ so 0.51. However, $\abs{0 - 1} = 1$, so 0 R̸ 1. Thus, R is not transitive.

Proving a Relation is an Equivalence Relation Suppose that the relation R on X is reflexive, and satisfies the “twisted transitive law”: ∀(x,y,z∈X)(xRy∧xRz⟹yRz) Show that R is an equivalence relation.

Proof. Let R be a relation on X that is reflexive and satisfies the “twisted transitive law”. We need to show that the relation is symmetric and transitive. Let x, y, z ∈ X be arbitrary.

  1. Assume x R y. Since R is reflexive, then x R x. Then by the “twisted transitive law”, y R x. Thus, R is symmetric.

  2. Assume x R y and y R z. Then y R x because R is symmetric. Applying the “twisted transitive law” to y R x and y R z gives x R z. Thus, R is transitive.

Therefore, R is an equivalence relation. ◻

Equivalence Class Suppose that ∼ is an equivalence relation on X. For each x ∈ X, the set $$\overline{x} = \left\{ y \in X : x \sim y \right\}$$ is called the equivalence class of x.

Distinct Equivalence Classes Let X be a set with equivalence relation ∼. If x, y ∈ X and $\overline{x} \cap \overline{y} \neq \emptyset$, then $\overline{x} = \overline{y}$.

Proof. Since $\overline{x} \cap \overline{y} \neq \emptyset$, then there exists some $w \in \overline{x} \cap \overline{y}$. That is, $w \in \overline{x}$ and $w \in \overline{y}$, so x ∼ w and y ∼ w. Suppose $z \in \overline{x}$. Then x ∼ z and z ∼ x. Since z ∼ x and x ∼ w, then by transitivity, z ∼ w. By symmetry, w ∼ z. From y ∼ w and w ∼ z we have y ∼ z by transitivity. Thus, $z \in \overline{y}$. Therefore, $\overline{x} \subseteq \overline{y}$. By a similar argument, we can also prove $\overline{y} \subseteq \overline{x}$, so $\overline{x} = \overline{y}$. ◻

If any two equivalence classes intersect each other, then they must be the same equivalence class. Two distinct equivalence classes are always disjoint.

If X is a set with equivalence relation ∼, $y \in \overline{x}$, then $\overline{x} = \overline{y}$.

Proof. Because ∼ is reflexive, we know that $y \in \overline{y}$. That is, $\overline{x} \cap \overline{y} \neq \emptyset$. Therefore, $\overline{x} = \overline{y}$ by the previous theorem. ◻

Partition A partition of a set X is a collection P of subsets of X such that:

  1. for all x ∈ X, there exists A ∈ P such that x ∈ A, and

  2. if A, B ∈ P and A ∩ B ≠ ∅, then A = B.

That is, a partition of X divides X into disjoint subsets that cover X. No two distinct subsets of a partition may contain any same elements.

Also note that no two equivalence classes of a set will contain any same elements. If we have an equivalence relation on X, then grouping each element of X into its respective equivalence class forms a partition of X. If X is a set with equivalence relation ∼, then the equivalence classes form a partition of X.

Given a nonempty set X, there are two “trivial” equivalence relations:

  1. x ∼ y if and only if x = y

    In this case, equivalence classes are the sets $\overline{x} = \{x\}$.

  2. x ∼ y for all x, y ∈ X

    In this case, there is only one distinct equivalence class where $\overline{x} = X$ for all x ∈ X.

For two sets A and B, define A ∼ B to mean that there is a bijection f : A → B. Show that ∼ is an equivalence relation.

Proof. For a set A, the function idA : A → A is a bijection, so ∼ is reflexive.

Suppose that A ∼ B. That is, there exists a bijection f : A → B. Then f−1 : B → A is a bijection, so B ∼ A. Thus, ∼ is symmetric.

Suppose that A ∼ B and B ∼ C. That is, there exists a bijection f : A → B and a bijection g : B → C. Then g ∘ f : A → C is a bijection. Thus, ∼ is transitive.

Therefore, ∼ is an equivalence relation. ◻

Modular Arithmetic

Modular Equivalence For m, n ∈ ℤ and k ∈ ℕ, we say that m is equivalent to n modulo k if k ∣ (m−n). $$\underbrace{m \equiv n \pmod{k}}_{\text{``$m$ is equivalent to $n$ mod $k$''}} \iff \underbrace{k \mid (m - n)}_{m-n = kc\ \text{for}\ c \in \mathbb{Z}}$$

Note that modular equivalence is an equivalence relation on ℤ.

  1. Reflexive: For m ∈ ℤ, m − m = 0 ⋅ k, so m ≡ m (mod  k)

  2. Symmetric: If m ≡ n (mod  k), then m − n = kc for some c ∈ ℤ. Thus, n − m = k(−c) where c ∈ ℤ, so n ≡ m (mod  k).

  3. Transitive: If l ≡ m (mod  k) and m ≡ n (mod  l), then l − m = ka and m − n = kb where a, b ∈ ℤ. That is: l − n = (l−m) + (m−n) = ka + kb = k(a+b) ∈ ℤ Therefore, l ≡ n (mod  k).

The set of equivalence classes modulo k is denoted ℤk.

ℤ2 In ℤ2, we will have two equivalence classes, $\overline{0}$ and $\overline{1}$. That is, $\mathbb{Z}_2 = \{\overline{0}, \overline{1}\}$ $$\begin{aligned} \overline{0} &= \{ x \in \mathbb{Z}: 2 \mid (x - 0) \} = \{ \ldots, -4, -2, 0, 2, 4, \ldots \} \\ \overline{1} &= \{ x \in \mathbb{Z}: 2 \mid (x - 1) \} = \{ \ldots, -3, -1, 1, 3, 5, \ldots \} \end{aligned}$$

Similarly, $\mathbb{Z}_3 = \{\overline{0}, \overline{1}, \overline{2}\}$.

$Z_k = \{\overline{0}, \ldots, \overline{k-1}\}$ For all k ∈ ℕ, $\mathbb{Z}_k = \{\overline{0}, \overline{1}, \ldots, \overline{k-1}\}$, and these equivalence classes are distinct.

Proof. Let m ∈ ℤ. By the division algorithm, we can write m = qk + r where q, r ∈ ℤ and 0 ≤ r ≤ k − 1. Then, m − r = qk, so m ≡ r (mod  k). That is $\overline{m} = \overline{r}$, so $\overline{m}$ is one of the classes $\overline{0}, \overline{1}, \ldots, \overline{k-1}$. Suppose for contradiction $\overline{m} = \overline{n}$ where 0 ≤ n < m ≤ k − 1. Because they are the same equivalence class, then k ∣ (m−n). However, k > m − n which contradicts k ∣ (m−n). Thus, the classes $\overline{0}, \overline{1}, \ldots, \overline{k-1}$ are all distinct. ◻

Moreover, for any k ∈ ℕ and m ∈ ℤ, we have $\overline{m} = \overline{0}$ in ℤk if and only if k ∣ m.

In ℤ6: $$\overline{0} = \{ \ldots, -12, -6, 0, 6, 12, \ldots\} = \{m \in \mathbb{Z}: 6 \mid m\}$$

  • $\overline{19} = \overline{1}$ because 19 = 3 ⋅ 6 + 1

  • $\overline{43} = \overline{1}$ because 43 = 7 ⋅ 6 + 1

  • $\overline{-38} = \overline{4}$ because  − 38 =  − 7 ⋅ 6 + 4

To perform arithmetic in ℤk, we need to define addition and multiplication.

Modular Addition For $\overline{x}, \overline{y} \in \mathbb{Z}_k$, modular addition is defined as: $$\overline{x} + \overline{y} \coloneq \overline{x+y}$$

Also, we need to check that these operations are well-defined.

This operation is well-defined if: $$\overline{x} = \overline{w}\ \land\ \overline{y} = \overline{z} \implies \overline{x+y} = \overline{w+z}$$ Since there are many elements we can choose to “represent” an equivalence class in our definition of Modular Addition, we need to ensure that every possible choice follows the definition.

If $\overline{x} = \overline{w}$, then x − w = ak for some a ∈ ℤ. Similarly, if $\overline{y} = \overline{z}$, then y − z = bk for some b ∈ ℤ. $$\begin{aligned} (x+y) - (w+z) &= (x-w) + (y-z) \\ &= ak + bk \\ &= (a+b)k, \quad \text{where}\ a+b \in \mathbb{Z} \end{aligned}$$ Thus, x + y ≡ w + z (mod  k), so $\overline{x+y} = \overline{w+z}$.

Similarly, we can show that $\overline{x} \cdot \overline{y} = \overline{x \cdot y}$ is well-defined.

For all k ∈ ℕ, the set ℤk with addition and multiplication satisfies the field axioms except possibly the existence of multiplicative inverses.

Proof. All we need to show is the existence of multiplicative inverses. Zero Product Property in Modular Arithmetic If p ∈ ℕ is prime, and a, b ∈ ℤ where $\overline{ab} = \overline{0}$ in ℤp, then $\overline{a} = \overline{0}$ or $\overline{b} = \overline{0}$.

Proof. If $\overline{ab} = \overline{0}$ in ℤp, then ab = kp for some k ∈ ℤ. Therefore, p ∣ ab, so p ∣ a or p ∣ b (Corollary 3.5.6).

  • If p ∣ a, then a = mp for some m ∈ ℤ. Thus, $\overline{a} = \overline{0}$.

  • If p ∣ b, then b = np for some n ∈ ℤ. Thus, $\overline{b} = \overline{0}$.

One of the above statements must be true. Therefore, $\overline{a} = \overline{0}$ or $\overline{b} = \overline{0}$. ◻

Restriction If f : X → Y is a function, and A ⊆ X, then the restriction of f to A is the function: $${ \left.\kern-\nulldelimiterspace f \vphantom{\big|} \right|_{A} } : A \to Y$$ defined by ${ \left.\kern-\nulldelimiterspace f \vphantom{\big|} \right|_{A} } = f(x)$ for all x ∈ A.

We can read this as “f restricted to domain A”.

If f : X → Y is injective and A ⊆ X, then ${ \left.\kern-\nulldelimiterspace f \vphantom{\big|} \right|_{A} }$ is injective.

Proof. Let $g \coloneq { \left.\kern-\nulldelimiterspace f \vphantom{\big|} \right|_{A} }$. Suppose there exist x1, x2 ∈ A such that g(x1) = g(x2). By definition of restriction, we have f(x1) = g(x1) and f(x2) = g(x2). Hence, f(x1) = f(x2). Since f is injective, then x1 = x2. Therefore, ${ \left.\kern-\nulldelimiterspace f \vphantom{\big|} \right|_{A} }$ is injective. ◻

Corestriction The corestriction of f is the function f : X → f(X).

In other words, the corestriction replaces the codomain with the range. As such, a corestriction is always onto. Although this is a different function, we typically use the same name.

Consider f : ℝ → ℝ defined by f(x) = x2. The corestriction of f is f : ℝ → [0, ∞).

If f is an injection, then the corestriction of f is a bijection. An important use of bijections is defining the cardinality of a set.

Cardinality The cardinality of a set is a measure of the amount of elements in that set.

For finite sets, cardinality follows our intuition of “size”. We can denote the cardinality of a set using the absolute value notation.

  • If A = ∅, then $\abs{A} = 0$.

  • If there exists a bijection f : A → {i ∈ ℕ : i ≤ n}, then $\abs{A} = n$.

  • If A and B are nonempty, and there exists a bijection f : A → B, then $\abs{A} = \abs{B}$

In these cases, we say A is finite with cardinality $\abs{A}$. We will write {i ∈ ℕ : i ≤ n} simply as {1, …, n}, and we can list a finite set as A = {a1, a2, …, an}.

Our intuition of cardinality as “size” can deceive us when dealing with infinite sets.

Cardinality of ℤ2 The sets ℤ and 2ℤ have the same cardinality.

Proof. Let f : ℤ → 2ℤ be defined by f(x) = 2x. f is a bijection, so $\abs{\mathbb{Z}} = \abs{2\mathbb{Z}}$. ◻

Although our intuition serves well in understanding cardinalities of finite sets, it can be difficult giving a logically sound explanation.

Cardinality of Subsets If A is finite and B ⊆ A, then B is finite and $\abs{B} \leq \abs{A}$.

Proof. If A = ∅ and B ⊆ A, then B = ∅. Thus, B is finite, and $\abs{A} = \abs{B} = 0$, so $\abs{B} \leq \abs{A}$.

Otherwise, A = {a1, …, an} for some n ∈ ℕ. We can then use induction. For our base case, suppose n = 1. Then $\abs{A} = 1$. If B ⊆ A, then either B = ∅ or B = A. Then $\abs{B} = 0$ or $\abs{B} = 1$ respectively. In either case, $\abs{B} \leq \abs{A}$.

Now, suppose the lemma is true for some n ∈ ℕ where n ≥ 1. Let A = {a1, …, an + 1}. By the induction hypothesis, there exists a bijection f : {a, …, an} → {1, …, n}. If B ⊆ A, then one of the following must be true:

  • B = ∅, so $\abs{B} = 0 \leq n+1 = \abs{A}$

  • B = A, so $\abs{B} = \abs{A}$

  • 0 ⊊ B ⊊ A, so there exists am ∈ A \ B, 1 ≤ m ≤ n + 1.

Let g : A → {1, …, n + 1} be a function defined as such:

  • g(aj) = f(aj) if j ≠ n + 1 and j ≠ m

  • g(am) = n + 1

  • g(an + 1) = f(am)

Thus, g is a bijection (exercise 4.22). Let $h \coloneq { \left.\kern-\nulldelimiterspace g \vphantom{\big|} \right|_{B} }$. Then h is a bijection, and h(B) ⊆ {1, …, n}. This is finite by the induction hypothesis, so $\abs{h(B)} \leq n$. But h : B → h(B) is a bijection, so $\abs{B} = \abs{h(B)} \leq n < \abs{A}$. Therefore, B is finite, and $\abs{B} \leq \abs{A}$. ◻

Partition Preserves Cardinality If A is a finite set and {A1, …, An} is a partition of A, then $\abs{A} = \sum_{i=1}^{n} \abs{A_i}$.

Proof. Let A be a finite set. We will use induction in this proof. For convenience, let P(n) be the statement $\abs{A} = \sum_{i=1}^{n} \abs{A_i}$. Note that A is finite and Ai ⊆ A for all i, so Ai is also finite.

We will first prove P(1). If A1 = A, and $\abs{A_1} = \abs{A} = \sum_{i=1}^{1} \abs{A_i}$.

Next, assume P(n) is true for some n ∈ ℕ. Let {A1, …, An + 1} be a partition of A. Let $B \coloneq \bigcup_{i=1}^{n}A_i$. Then {A1, …, An} is a partition of B. By the induction hypothesis, $\abs{B} = \sum_{i=1}^{n} \abs{A_i}$. Note that B and An + 1 is disjoint (otherwise, our collection of A1 to An would not be a partition). Also note that A = B ∪ An + 1. Thus, by the : $$\begin{aligned} \abs{A} &= \abs{B} + \abs{A_{n+1}} \\ &= \sum_{i=1}^{n} \abs{A_i} + \abs{A_{n+1}} \\ &= \sum_{i=1}^{n+1} \abs{A_i} \end{aligned}$$ Therefore, by the , P(n) is true for all n ∈ ℕ (so long as we can partition A into n sets). ◻

Pigeonhole Principlepigeonhole Let f : X → Y be a function where $\abs{X} > \abs{Y} > 0$. Then there exist x1, x2 ∈ X where f(x1) = f(x2) but x1 ≠ x2.

Intuitively, we can think of the elements of X as pigeons and the elements of Y as pigeonholes. Our function f tries to put each pigeon into a pigeonhole. If there are more pigeons than there are pigeonholes, then at least two pigeons will share a pigeonhole.

Prove that if A ⊆ {1, 2, …, 9} where $\abs{A} = 6$, then there are distinct elements x, y ∈ A such that x + y = 10.

Proof. Let B ≔ {{1, 9}, {2, 8}, {3, 7}, {4, 6}, {5}}. Then $\abs{B} = 5$. Let f : A → B be a function defined as follows: “for each x ∈ A, let f(x) be the unique element of B that contains x.” Note that f is a function between finite sets where $\abs{A} > \abs{B} > 0$. Therefore, by the Pigeonhole Principle, there exist x, y ∈ A such that x ≠ y and f(x) = f(y). That is, x and y are in the same set in B, so x + y = 10. ◻

In the above example, we can think of A as our pigeons and B as our pigeonholes.

We know from previous examples that ℤk satisfies every field axiom except existence of multiplicative inverses if k is not prime. We can prove that if k is prime and $\overline{m} \in \mathbb{Z}_k$ and $\overline{m} \neq \overline{0}$, then $\overline{m}$ has a multiplicative inverse in ℤk.

Conditions for ℤk being a field For k ∈ ℕ where k ≥ 2, ℤk is a field if and only if k is prime.

Proof. Fix $\overline{m} \in \mathbb{Z}_k$ where $\overline{m} \neq \overline{0}$, and define the function f : ℤk → ℤk as $f\left(\overline{n}\right) = \overline{mn}$. If $f\left(\overline{a}\right) = f\left( \overline{b} \right)$ for $\overline{a}, \overline{b} \in \mathbb{Z}_k$, then $\overline{ma} = \overline{mb}$. Thus: $$\overline{0} = \overline{ma} - \overline{mb} = \overline{m} \left( \overline{a} - \overline{b} \right)$$ Because k is prime, either $\overline{m} = \overline{0}$ or $\overline{a} - \overline{b} = \overline{0}$. But $\overline{m} \neq \overline{0}$, so $\overline{a} - \overline{b} = 0$. That is, $\overline{a} = \overline{b}$, so f is one-to-one. By Corollary 4.3.9, f is also onto, making f a bijection. Thus, there exists $\overline{c} \in \mathbb{Z}_k$ such that $f \left( \overline{c} \right) = \overline{1}$ That is, $\overline{1} = f \left( \overline{c} \right) = \overline{mc} = \overline{m} \cdot \overline{c}$, so $\overline{m}^{-1} = \overline{c}$ in ℤk. Therefore, if k is prime, then every $\overline{m} \in \mathbb{Z}_k$ where $\overline{m} \neq \overline{0}$ has a multiplicative inverse ℤk, so ℤk is a field. ◻

Infinite Primes (Revisited) The set of all prime numbers is infinite.

Proof. Suppose for contradiction that there exist only a finite number of primes. We can list these primes as p1, p2, …, pk. Let n ≔ p1p2⋯pk + 1. By the Division Algorithm, for all i ∈ ℤ where 1 ≤ i ≤ k, we can write n = qpi + r where q ∈ ℤ and 0 ≤ r < pi. Moreover, q and r are unique. Thus: $$q = \frac{p_1 p_2 \cdots p_k}{p_i} \quad \text{and} \quad r = 1$$ If pi ∣ n for some i, then n = spi + 0 for some s ∈ ℤ. This contradicts the uniqueness of q and r guaranteed by the Division Algorithm. Hence, none of the primes p1, p2, …, pk divide n. This contradicts n having a prime factorization. Therefore, there must exist infinitely many primes. ◻

Countably Infinite A set is countably infinite if that set has the same cardinality as ℕ.

Uncountably Infinite A set is uncountably infinite if that set is infinite but not countably infinite.

We will categorize all finite and countably infinite sets as as “countable” sets. We will also categorize all countably infinite and uncountably infinite sets as “infinite” sets.

To help our intuition, we can think of a countable set as having an idea of a “next element”. In ℕ, we can list 1, 2, 3, and so on. However, if we start listing real numbers from 0, what is the “next” real number?

Show that if A is countable and f : A → B is a bijection, then B is countable.

Proof. Since A is countable, then A is either finite or countably infinite.

  • If A is countably infinite, then there exists a bijection g : A → ℕ. Again, g ∘ f−1 : B → ℕ is a bijection. Thus, B is countably infinite.

In either case, B is countable (finite or countably infinite). ◻

Cantor’s Diagonal Argumentcantor-diagonal The set [0,1] ⊂ ℝ is uncountable.

Proof. We will use the idea that every real number in [0,1] has a decimal expansion 0.δ1δ2δ3… where each δ represents a digit of the number. Conversely, every decimal expansion 0.δ1δ2δ3… represents a real number in the interval [0,1] ∈ ℝ.

This means we can “list” all the real numbers in the interval [0,1]. $$\begin{aligned} 1 &\mapsto 0.\delta_{11} \delta_{12} \delta_{13} \ldots \\ 2 &\mapsto 0.\delta_{21} \delta_{22} \delta_{23} \ldots \\ 3 &\mapsto 0.\delta_{31} \delta_{32} \delta_{33} . \ldots \\ & \vdots \end{aligned}$$ We can now construct a real number that is not in this list.
